LISTAS Y ENLACES EN HTML5


Las listas son uno de los elementos mas comunes del html permiten organizar la información con viñetas, letras o números, existen 3 tipos de listas(numeradas, no numeradas y de definicion), vamos explicar los 2 tipos de listas mas utilizados las numeradas y las no numeradas.

listas numeradas

Tambien conocida como listas ordenadas, es una etiqueta que se utiliza para llevar un orden bien sea numérico o alfanumérico, al momento de listar una información. Se identifica con la etiqueta ol

Ejemplo de la estructura de la lista ordenada

listas no numeradas

Tambien conocida como listas desordenadas, es una etiqueta que se utiliza para llevar una lista de elementos con viñetas, esta ee identifica con la etiqueta Ul

Ejemplo de la estructura de la lista desordenada

Enlaces HTML

Son las etiquetas que nos permiten comunicarnos con cualquier recurso dentro del mismo directorio o con otro directorio
